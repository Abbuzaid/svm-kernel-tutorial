## Key Findings
- RBF kernel — best general-purpose choice
- Linear kernel — fastest, best for high-dimensional data
- Polynomial kernel — tunable complexity via degree
- Sigmoid kernel — generally not recommended
- Always standardise features and tune C and gamma
